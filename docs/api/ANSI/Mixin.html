Mixin

This module is designed specifically for mixing into String-like classes or extending String-like objects.

Generally speaking the String#ansi method is the more elegant approach to modifying a string with codes via a method call. But in some cases this Mixin’s design might be preferable. In particular it was primarily designed to provide a compatability layer for the colored gem.

def display(line, column=0)
  result = "\e[s"
  result << "\e[#{line.to_i};#{column.to_i}H"
  result << to_s
  result << "\e[u"
  result
end
